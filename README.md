# Homebrew Tap

Homebrew formulae for [sflinter](https://github.com/sflinter) projects.

## Install

```sh
brew tap sflinter/tap
brew install banktivity-swift-mcp
```

## Formulae

| Formula | Description |
|---|---|
| `banktivity-swift-mcp` | MCP server and CLI for Banktivity personal finance vaults |
